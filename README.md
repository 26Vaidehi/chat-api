# Real-Time Chat API

A full-stack real-time chat application built with Node.js, Socket.io, and MongoDB.

## 🔗 Live Demo
[https://chatmesh-yklw.onrender.com](https://chatmesh-yklw.onrender.com)


## 🛠 Tech Stack
- **Backend:** Node.js, Express.js
- **Real-Time:** Socket.io
- **Database:** MongoDB Atlas
- **Auth:** JWT, bcryptjs
- **Deployment:** Render

## ✨ Features
- User registration and login with bcrypt password hashing
- JWT-based authentication
- Real-time messaging with Socket.io
- Multiple chat rooms
- Message history stored in MongoDB
- Responsive frontend

## 🚀 Run Locally
```bash
git clone https://github.com/26Vaidehi/chat-api.git
cd chat-api
npm install
# Add .env file with PORT, JWT_SECRET, MONGO_URI
npm run dev
```
